# @journal-formatter — Workflow & Responsibilities

> **Role:** Ensure 100% adherence to target journal formatting guidelines
> **Active:** Phase 5
> **Tier:** 4 — Writing, Languages & Submission

---

## Mission

Zero formatting-related desk rejects. Every margin, citation style, figure placement, and word count must comply perfectly.

---

## Journal A — The Lancet Planetary Health

| Element | Requirement |
|---------|------------|
| Word count (body) | 4,500 max (excl. abstract, refs, tables, figures) |
| Abstract | Structured, 300 words: Background, Methods, Findings, Interpretation, Funding |
| Figures + Tables | Max 5 combined (additional in supplement) |
| References | Vancouver style (numbered), max 40 |
| Figure format | EPS, PDF, or TIFF; min 300 DPI; max width 180mm |
| Supplementary | Appendix with additional tables, figures, methods |
| STROBE checklist | Required (observational study) |
| Cover letter | Must state novelty and significance |
| Conflict of interest | Signed ICMJE forms |
| Data statement | Data availability statement required |
| Ethics | Ethics statement (or exemption for ecological study) |
| File format | Word (.docx) for submission; LaTeX → Word conversion needed |

## Journal B — Environmental Health Perspectives

| Element | Requirement |
|---------|------------|
| Word count | 6,000 max |
| Abstract | Unstructured, 250 words |
| Figures + Tables | Max 6 combined |
| References | No strict limit |
| Code/data | GitHub repository required |
| STROBE | Recommended |

## Journal C — Environment International

| Element | Requirement |
|---------|------------|
| Word count | 8,000 max |
| Graphical abstract | Required |
| Figures/Tables | No strict limit |
| References | No strict limit |
| Highlights | 3-5 bullet points (85 chars each) |
